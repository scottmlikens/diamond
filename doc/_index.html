Bootstrapping instances

We currently deploy 10.24.0 on our servers in production. You can bootstrap them like:
shell knife bootstrap 74.201.193.100 -N marketplace-176852 --bootstrap-version 10.24.0 --environment production -r "role[marketplace]" -x ubuntu --sudo

Old getting Started Information

2) Ask Nafis or James Sulinksi to add you to the mopub organization on opscode

3) Accept the invitation to the account

4) Get private key from opscode account page and place it in ~/.chef

5) Get chef.pem file and put in keychain

6) chmod 0600 chef.pem

7) Get validation key and put in ~/.chef/

8) on command line: brew install ruby

9) add ruby path to .bash_profile and source it

10) on command line: sudo gem install chef knife-ec2

11) git clone git@github.com:mopub/mopub-chef.git

12) Then you need to export environment variables that the knife.rb file uses to your bash_profile:

export OPSCODE_USER=
export AWS_ACCESS_KEY_ID=
export AWS_SECRET_ACCESS_KEY=
export AWS_SSH_KEY_ID=chef
export ORGNAME=mopub

13) Source bash_profile or open a new shell

14) knife node list (to test)
